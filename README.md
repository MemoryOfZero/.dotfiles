## My dots for Arch/Manjaro using bspwm + polybar

For zsh:
Create a new .zshenv in ~/ with the following: 
```
export ZDOTDIR=~/.dotfiles/zsh
source $ZDOTDIR/.zshenv
```

Symlink everything else

- bspwm
- dmenu
